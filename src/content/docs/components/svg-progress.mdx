### `o-svg progress` (SVG)

The `svg progress` component also works with orbits and serves the purpose of creating circular progress bars or knobs. It is implemented using SVG.

```html
   <div class="orbit-3 ">
      <svg class="svg-pod">
        <circle class="svg-progress" style="--val: 59;" /> 
      </svg>
    </div>
```
