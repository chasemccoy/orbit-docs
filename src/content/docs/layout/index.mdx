Orbit is a CSS framework designed for creating radial layouts. It offers simplicity, effectiveness, ease of use, and tremendous versatility for crafting compelling designs. It offers extensive customization options and supports nesting for building intricate designs tailored to your requirements.

### Step 1: Adding a Bigbang
**[Bigbang](/orbit-docs/elements/bigbang)** class is a wrapper for your Orbit project. Only one Bigbang is needed for each Orbit porject, or Orbit instance if you have multiples Orbits in same page. 

```html wrap
<div class="bigbang">
    
</div>
```

### Step 2: Adding a Gravity Spot
A **[Gravity Spot](/orbit-docs/elements/gravity-spot)** is a container for orbits. Each Gravity Spot has a default size of 500px but can be easily adjusted to fit your project's needs. Inside an Gravity Spot, radial elements will adhere to this size. For example, the maximum diameter of an orbit within this spot will be 500px.

Metaphorically, a gravity spot is a gravitational center, and its size (500px), refers to its max gravitational force. 

```html wrap
<div class="bigbang">
    <div class="gravity-spot">
    </div>
</div>
```

### Step 3: Adding Orbits
Within the Gravity Spot container, there are three ways to add **[orbits](/orbit-docs/elements/orbit)**: 1. Manually set orbit radius ranging from `orbit0` to `orbit12`; 2. Leave Orbit determine orbits sizes using `.orbit`; 3. By mixing 1 and 2.


```html wrap
<div class="bigbang">
    <div class="gravity-spot">
        <div class="orbit"></div>
        <div class="orbit"></div>
        <div class="orbit"></div>
        <div class="orbit"></div>
        <div class="orbit-6"></div>
    </div>
</div>
```

### Step 4: Adding Radial Elements
Each orbit can contain **radial elements** such as [`satellites`](/orbit-docs/elements/satellite),[ `slices`](/orbit-docs/elements/slice), [`vectors`](/orbit-docs/elements/vector), [`curved texts`](/orbit-docs/elements/text),[ `progress`](/orbit-docs/elements/progress) indicators, and [`sides`](/orbit-docs/elements/side). These elements are evenly distributed along the orbit's perimeter, allowing for detailed radial designs tailored to specific needs.

```html wrap
<div class="bigbang">
    <div class="gravity-spot">
        <div class="orbit">
             <div class="satellite">
                <div class="capsule">volume</div>
             </div>
        </div>
        <div class="orbit-4"></div>
        <div class="orbit-5">
            <div class="vector"></div>

            <div class="vector"></div>

            <div class="vector"></div>
            <div class="vector"></div>
            <div class="vector"></div>
            <div class="vector"></div>
            <div class="vector"></div>
            <div class="vector"></div>
        </div>
        <div class="orbit-6"></div>
        <div class="orbit-6">
            <o-progress value="50"></o-progress>
        </div>
    </div>
</div>
```

**Gravity Spot** can be nested within `satellites`, enabling the creation of complex radial structures.

    <div class="gravity-spot">

### Step 6: Customizing
There are numerous `CSS class utilities` to use with all Orbit elements. For instance, to distribute satellites within a range of 180 degrees, you can use the `.range-180` class on the parent orbit element. 

In the example below, we used these CSS Utilities: `.at-center`, `.shrink-*`, `.from-*`, `.range-*`, and `.outer-orbit`
