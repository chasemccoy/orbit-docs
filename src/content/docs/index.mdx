<Card title="Simplifying radial UI design" >
Orbit is a CSS library designed to make radial UI designs simple and accessible. Perfect for creating everything from circular menus to intricate data visualizations, Orbit provides an easy-to-use solution for your creative needs.
</Card>
	


## Key features

<CardGrid>
	<Card title="Ease of use" icon="pencil">
		Designed for simplicity, making radial design accessible to all developers.
	</Card>
	<Card title="Flexible integration" icon="add-document">
		Works standalone or alongside other CSS frameworks.
	</Card>
	<Card title="Versatile applications" icon="setting">
		Intuitive class system for quick implementation. From menus to visualizations, Orbit adapts to various needs.
	</Card>
	<Card title="Pure CSS" icon="open-book">
	    A JavaScript-free approach for simpler, more flexible designs.
	</Card>
</CardGrid>
