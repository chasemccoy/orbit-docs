# Orbit CSS Framework

**Orbit** is a CSS framework designed to create complex circular UI designs effortlessly. It provides intuitive CSS classes and vanilla web components for building circular menus, data visualization components, and circular interfaces.

## Installation

### Download or Include via CDN

#### CSS
To include Orbit's CSS in your project, you have three options:

1. **Download the CSS file**:
   - Download the CSS file from the repository.
   - Link it in your HTML file:
     ```html
     <head>
       <link rel="stylesheet" href="path/to/orbit.css">
     </head>
     ```

2. **Use the `@import` rule**:
   - Add the following to your stylesheet:
     ```css
     @import url('path/to/orbit.css');
     ```

3. **Include via jsDelivr CDN**:
   - Add the following to your HTML file:
     ```html
     <head>
       <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/@zumerlab/orbit/orbit.css">
     </head>
     ```

#### JavaScript
To include Orbit's JavaScript for web components, you have two options:

1. **Download the JavaScript file**:
   - Download the JS file from the repository.
   - Link it in your HTML file:
     ```html
     <head>
       <script src="path/to/orbit.js" defer></script>
     </head>
     ```

2. **Include via jsDelivr CDN**:
   - Add the following to your HTML file:
     ```html
     <head>
       <script src="https://cdn.jsdelivr.net/npm/@zumerlab/orbit/orbit.js" defer></script>
     </head>
     ```

### Install via Package Manager
You can also install Orbit using npm or yarn:

```sh
npm install @zumerlab/orbit
```
